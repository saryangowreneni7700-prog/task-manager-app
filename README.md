# Task Manager App

Full Stack Task Manager Application built using MERN stack.

## 🚀 Features
- Add tasks
- Delete tasks
- Mark tasks as completed
- Full Stack Architecture

## 🛠️ Tech Stack
- Frontend: React.js
- Backend: Node.js, Express.js
- Database: MongoDB

## ▶️ How to Run

### Backend
cd server  
npm install  
node server.js  

### Frontend
cd client  
npm install  
npm start
